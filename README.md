# Sistema de Gestão de Leilões

## Sobre o Projeto
Sistema desenvolvido em Java para informatizar as rotinas de uma
casa de leilões, incluindo cadastro de lotes, clientes, lances e leilões.

## Tecnologias Utilizadas
- Java (JDK 17)
- MySQL (banco de dados relacional)
- JDBC (conexão Java com MySQL)
- Git / GitHub (versionamento)
